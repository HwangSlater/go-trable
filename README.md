# 여행 모임 컨벤션 (Draft)

## 1. 여행 기본 방향
- 목적: 액티비티 + 관광 + 휴식
- 컨셉: **1박 2일 감성 + 컨텐츠 즐기기**
- 저녁 시간은 휴식 및 자유 시간으로 운영
- 비 오는 날도 감성 요소로 인정
  
---

## 2. 일정 원칙
- 기본 일정: **2박 3일**
- 날짜 후보:
  - 5/22 ~ 5/24
  - 5/23 ~ 5/25
- 공휴일 활용 전제
- 평일 연차 다수 사용 일정은 현실적으로 지양

---

## 3. 집합 방식
- **금요일 밤 집합**
  - 목표 지역에 숙소를 미리 예약
  - 각자 퇴근 후 이동
  - 금요일 밤: 모이는 것 자체가 컨텐츠
  - 가볍게 술 + 휴식
- 토요일 아침부터 본격 일정 시작

---

## 4. 숙소 규칙
- 숙소는 **목표 지역 기준으로 1곳 예약**
- 예약 시점:
  - 최소 **1달 전 예약**
- 숙소 예약 이후:
  - 개인 사정으로 불참 시 **회비 환불 불가**
  - 예외 없음

---

## 5. 참여 원칙
- 일정 확정 전: 자유 의사 표현 가능
- 숙소 예약 후: 참여 확정으로 간주
- 프로젝트/개인 일정으로 인한 불참 가능성은 **사전 공유 필수**

---

## 7. 기타
- 일정 중 날씨 변수 발생 시:
  - 일정 일부 조정 가능
  - 감성 요소로 수용
